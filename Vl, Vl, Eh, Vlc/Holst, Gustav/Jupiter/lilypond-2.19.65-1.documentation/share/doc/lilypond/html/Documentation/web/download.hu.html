Jogi tudnivalók

Minden logó és termékábrázolás szerzői joggal védett.

Tux, a Linux pingvin lewing@isc.tamu.edu műve, és a GNU Image Manipulation Program segítségével készült.

A FreeBSD logó a FreeBSD Alapítvány bejegyzett védjegye.

Az „X-et ábrázoló kép” nem bejegyzett védjegy. Nihonjoe és CyberSkull műve, a GNU Free Documentation License (GNU Szabad Dokumentációs Licenc) 1.2-es verziója vonatkozik rá. A kép forrása ez a Wikipedia Commons oldal.

A „négyszínű ablakot ábrázoló kép” nem bejegyzett védjegy. Rohitbd műve és a GNU Free Documentation License (GNU Szabad Dokumentációs Licenc) 1.2-es verziója vonatkozik rá. A kép forrása ez a Wikipedia Commons oldal.
