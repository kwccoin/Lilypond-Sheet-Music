Vývoj pro LilyPond 2.19.65

Pozor: Zde se jedná o nestálé vývojářské verze. Pokud nevíte přesně, jak se LilyPond instaluje a používá, důrazně doporučujeme používat verze stálé Stažení a číst Příručky.

Čísla uveřejnění

Jsou dvě zveřejňovací řady pro LilyPond: stálé verze a nestálé vývojářské verze. Stálé verze mají sudé druhé číslo verze (čili zhruba 2.8, 2.10, 2.12). Vývojářské verze mají druhé číslo verze liché (tedy 2.7, 2.9, 2.11).
