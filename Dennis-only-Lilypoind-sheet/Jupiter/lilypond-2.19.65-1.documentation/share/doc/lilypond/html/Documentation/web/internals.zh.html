内部机制参考手册 Internals reference

这是一个交叉链接页面的集合，这些页面把每一个 LilyPond 的类，对象和函数的本质细节进行了归档。它是从源码中的格式化定义直接生成的。

几乎所有调整格式的内部函数，用户可以直接使用。例如，绝大部分控制粗细值，距离等的变量，可以在乐谱输入文件中进行更改。LilyPond 有大量格式化选项，所有这些选项都在这份文档中进行了描述。Notation reference (记谱法参考手册) 的各个章节都有一个 See also (参见) 小节 ，指向着一份自动生成的文档。
