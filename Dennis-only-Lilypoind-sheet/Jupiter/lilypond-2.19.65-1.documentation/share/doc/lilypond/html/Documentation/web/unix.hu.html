Generikus vagy disztribúció-specifikus csomagokat használjak?

Sok disztribúció csomagkezelőjében megtalálható a LilyPond. Ezeket a verziókat egyszerűbb telepíteni és eltávolítani, mint a generikus csomagokat, de régebbiek lehetnek. Ha egy generikus csomagot akarsz használni, előbb távolítsd el a csomag hivatalos verzióját a csomagkezelővel. Ehhez a disztribúció csomagkezelőjének dokumentációja nyújt segítséget.

Generikus csomagok

Letöltés

  • GNU/Linux x86: LilyPond 2.18.2-1 (ha bizonytalan vagy afelől, hogy melyiket töltsd le, ezt válaszd)
  • GNU/Linux 64: LilyPond 2.18.2-1
  • GNU/Linux PPC: LilyPond 2.18.2-1
  • FreeBSD i386: LilyPond 2.18.2-1
  • FreeBSD amd64: LilyPond 2.18.2-1

Telepítés

Add ki a következő parancsokat: 

cd PATH-TO-DOWNLOAD-DIRECTORY
sh lilypond-2.18.2-OS-TYPE.sh

Eltávolítás

Add ki a következő parancsot: 

uninstall-lilypond

Disztribúció-specifikus csomagok

Ezeket a csomagokat a disztribúciód csomagkezelőjével tudod telepíteni.

  • Fedora: LilyPond 2.12.0
  • Ubuntu: LilyPond 2.12.1
  • Slackware: LilyPond 2.12.1
  • Debian: LilyPond 2.10.33
  • openSUSE: LilyPond 2.10.33
